### git指令
1. 拉代码
```
git pull
```
2. 查看修改
```
git status
```
3. 提交修改过的文件到缓存区
```
git add .
git add -u +/dir
git add -A +/dir
```
4. 提交到本地仓库
```
git commmit -m +"message"
```
5. 推送缓存区代码到服务器
```
git push
```
6. 撤回缓存区提交
```
git reset
```
7. 回退版本
```
git reset --hard +version
```
